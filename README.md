# 💫 Proyecto Final: Gestión para Boutique "Moda Viva"  

**Carrera:** Ingeniería en IA  
**Semestre:** 1° - 2025  

## 🌟 Sistema de gestión para tiendas de moda  

Solución completa para automatizar las operaciones diarias de una boutique, que incluye:  

- 🛍️ Registro de ventas intuitivo  
- 📦 Control de inventario en tiempo real  
- 📊 Reportes automáticos de gestión  

## 🔍 Qué encontrarás aquí  

- Código funcional en C++ con módulo demostrativo de búsqueda de productos  
- Documentación con casos de uso realistas  
- Diseño centrado en la experiencia de usuarios (vendedores y administradores)
